Drupal module that adds new field type, polish REGON identification number. It supports 9-digit and 14-digit long REGON numbers.

REGON (from Rejestr Gospodarki Narodowej â€“ Register of the National Economy) is a unique number granted to businesses in Poland. Sample REGON numbers can be generated [here](http://bogus.ovh.org/generatory/regon.html).

To use this module create a new node type and add new field REGON.
